##########################
Prerequisitos de Software
##########################

Para poder programar un chasis tipo **Swerve**, tienes que tener ciertos conocimientos previos, ya que este chasis tiene MUCHO de donde abarcar.
Estos conocimientos no se basan en fundamentos de programación, sino que son conocimientos en programación avanzada orientada a la robótica y las librerías como la **WPILIB**.

Command based programming
*************************

Esto se refiere a algo que va más allá de un **TimedRobot** común, ya que esto es una arquitectura, un framework completo que 
brinda grandes apoyos para una mayor organización del código, además de que facilita mucho la legibilidad y comprensión de cómo
se maneja cada parte del robot.

El artículo de la **WPILIB** sobre este tema es el siguiente: `Command Based Programming <https://docs.wpilib.org/en/latest/docs/software/commandbased/index.html>`_
De igual manera, tenemos documentación propia sobre este tema.
