#############################
Introducción a Advantage Kit
#############################

**Advantage Kit** es una librería de software de código abierto desarrollada por Mechanical Advantage,
diseñada para complementar a la WPILIB en el aspecto de logging y telemetría, facilitando la depuración
y el monitoreo del rendimiento del robot durante pruebas o partidas reales (incluso facilitando en simulaciones).

- El link de la documentación oficial es el siguiente: `Advantage Kit Documentation <https://docs.advantagekit.org/>`_

- El repositorio en GitHub es el siguiente: `Advantage Kit GitHub <https://github.com/Mechanical-Advantage/AdvantageKit>`_

- La api en java es el siguiente: `Advantage Kit Java API <https://docs.advantagekit.org/javadoc/>`_

Cabe aclarar que esta librería es opcional, pero es altamente recomendada para tener un mejor control
sobre todo el tema de logging.
Ahora si, a pesar de que toda la documentación oficial está en inglés, al igual que la información 
de la WPILIB, esta no suele ser amigable en cuanto a nuevos programadores, por lo que vamos a dar énfasis
a varios puntos importantes y dar ejemplos prácticos para que puedan entender mejor el uso de esta librería
en un código de FRC real.

.. important:: No vamos a traducir la librería, simplemente vamos a explicar lo que nosotros usamos de ella, pero siempre vamos a dar la referencia de ciertos temas para que puedan consultarlo desde la documentación oficial
