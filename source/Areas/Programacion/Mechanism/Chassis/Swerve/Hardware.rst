######################
Requisitos de hardware
######################

Para poder implementar un chasis tipo **Swerve**, se necesitan los siguientes componentes de hardware:

Módulos
*******

Swerve Drive Specialities (SDS):
================================
SDS es una marca que ofrece los siguientes módulos:

- `MK4 Module <https://www.swervedrivespecialties.com/products/mk4-swerve-module>`_

- `MK4i Module <https://www.swervedrivespecialties.com/products/mk4i-swerve-module>`_

- `MK4n Module <https://www.swervedrivespecialties.com/products/mk4n-swerve-module>`_

- `MK4c Module <https://www.swervedrivespecialties.com/products/mk4c-swerve-module>`_

- `MK5i Module <https://www.swervedrivespecialties.com/products/mk5i-swerve-module>`_

- `MK5n Module <https://www.swervedrivespecialties.com/products/mk5n-swerve-module>`_

West Coast Products (WCP):
==========================
WCP es una marca que ofrece los siguientes módulos:

- `X2 Module <https://wcproducts.com/collections/gearboxes/products/swerve-x2>`_

- `X2c Module <https://wcproducts.com/collections/gearboxes/products/swerve-x2c>`_

- `X2i Module <https://wcproducts.com/collections/gearboxes/products/swerve-x2i>`_

- `X2t Module <https://wcproducts.com/collections/gearboxes/products/swerve-x2t>`_

- `X2S Module <https://wcproducts.com/collections/gearboxes/products/swerve-x2-s>`_

- `X2St Module <https://wcproducts.com/collections/gearboxes/products/swerve-x2st>`_

Rev Robotics:
=============
Rev Robotics es una marca que ofrece los siguientes módulos:

- `3in MAXSwerve Module <https://www.revrobotics.com/rev-21-3005/>`_

- `4in EasySwerve Module <https://www.revrobotics.com/rev-21-3006/>`_

Casero
======
Un modulo casero no requiere de una marca, solo que:

- Tu imaginación Module

    - Cuesta tiempo y suele llegar a ser más caro si no se tiene experiencia.

    - Las especificaciones físicas las estableces tú.

    - Todas las características dependen de ti.

Motores
*******

Para cada módulo se necesitan dos motores, uno para mover la rueda y otro para girar el módulo.

Cross The Road Electronics (CTRE):
==================================
CTRE es una marca que ofrece los siguientes motores, estos se suelen usar para los módulos de Swerve de SDS y WCP:

- `Krakenx60 <https://store.ctr-electronics.com/products/kraken-x60>`_

- `Krakenx44 <https://store.ctr-electronics.com/products/kraken-x44>`_

Rev Robotics (REV):
===================
Rev Robotics es una marca que ofrece los siguientes motores, estos se suelen usar para los módulos de Swerve de Rev Robotics:

- `Neo Vortex <https://www.revrobotics.com/rev-21-1652/>`_

- `Neo v1.1 <https://www.revrobotics.com/rev-21-1650/>`_

- `Neo 550 <https://www.revrobotics.com/rev-21-1651/>`_

El detalle con los motores de Rev Robotics, es que los motores no vienen con controlador incluido, por lo que, se tiene que comprar:

- `Spark MAX <https://www.revrobotics.com/rev-11-2158/>`_

- `Spark Flex <https://www.revrobotics.com/rev-11-2159/>`_

De igual manera, rev ya vende como kits que incluyen el motor y el controlador.

Encoders Absolutos
******************
Los controladores de los motores suelen traer un encoder relativo integrado, pero para el giro del módulo se necesita uno absoluto para saber la posición exacta del módulo.

CTRE:
=====

- `CANCoder <https://store.ctr-electronics.com/products/cancoder?srsltid=AfmBOoqmH_Rz3yUln0vVx3BFSFbqiukmAMNzxOfMRHmaLqMw0glpolVY>`_
    
    - El cancoder es algo caro, pero vale la pena.

Rev Robotics:
=============

- `Through Bore Encoder <https://www.revrobotics.com/rev-11-1271/>`_

Giroscopio
**********

El giroscopio es un sensor que permite medir la orientación del robot, es decir, hacia dónde está apuntando el chasis.

- `Pigeon IMU <https://store.ctr-electronics.com/products/pigeon-2?srsltid=AfmBOorIphF2JxvLszxcC3AeJZ7ykJrPGMUrL264LeUxyuWNMpOKz8-u>`_

- `NavX-MXP <https://andymark.com/products/navx2-mxp-robotics-navigation-sensor>`_
